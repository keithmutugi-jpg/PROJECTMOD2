# Keith Tech Store

A simple front-end project built with HTML, vanilla CSS, and JavaScript.

## Main Features

- Responsive layout for mobile, tablet, and desktop
- Product data loaded from the public Fake Store API
- Complementary color palette with three main colors
- Three font families from Google Fonts
- Unit tests for JavaScript helper functions
- GitHub Actions workflow for GitHub Pages deployment

## File Structure

- `index.html` handles the homepage layout
- `about.html` explains the project approach
- `style.css` contains all styling and responsive rules
- `scripts/app.js` starts the app and connects the UI
- `scripts/api.js` fetches data from the public API
- `scripts/ui.js` renders product cards and updates the page
- `scripts/utils.js` stores reusable helper functions
- `tests/utils.test.js` contains unit tests

## Run Tests

```bash
npm test
```
