Project Overview

A small project built to stay clean, responsive, and easy to explain.

The goal of this site is to show a simple real-world front-end project using live data, structured styling, and a deployment workflow.

Three Fonts

Montserrat is used for headings, Open Sans for body text, and Space Mono for labels and stats.

Three Main Colors

The design centers on deep blue, warm orange, and soft cream to keep the palette controlled.

Vanilla CSS

No framework was used, so the layout and responsiveness come directly from CSS grid and flexbox.

How Data Works

API Fetching

The browser requests products from Fake Store API and then displays them as cards.

How Testing Works

Unit Tests

Pure helper functions are tested because they are the easiest parts to verify automatically.

How Deployment Works

GitHub Actions

Every push to `main` can run tests and deploy the static site to GitHub Pages.
